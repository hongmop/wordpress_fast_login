wordpress_fast_login
====================

安装插件后无需申请可以直接使用QQ、微博、人人帐号登录网站。

=============================================================
= 安装注意事项：
=   1、安装插件目录为wp-fast-login。
=   2、下载zip压缩文件后可以在wordpress后台直接安装。
=   3、安装完成后必须要先备置后才能正常使用。
==============================================================
